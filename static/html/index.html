API ENDPOINTS

Users Table

HTTP Methods Path Description
GET /user Retrieves a list of all users.
GET /user/:id Retrieves details of a specific user by ID.
POST /user/register Add a new user.
POST /user/login User is able to login.
PATCH /user/:id Updates details of a specific user by ID.
DELETE /user/:id Removes a user account by ID.

Foods Table

HTTP Methods Path Description
GET /food Retrieves a list of all food.
GET /food/recent Retrieves the 4 most recent items.
GET /food/:id Retrieves details of a specific food by ID.
POST /food/add Add a new food.
PATCH /food/:id Updates details of a specific food by ID.
DELETE /food/:id Removes a food by ID.

Orders Table

HTTP Methods Path Description
GET /user/:uid/orders Retrieves a list of all orders.
GET /user/:uid/order Retrieves all the orders of a particular user.
POST /user/:uid/order A user is able to add a order.
PATCH /user/:uid/order/:orderID An admin is able to update order.
DELETE /user/:uid/order/:orderID User is able to remove a single order.
DELETE /user/:uid/orders User is able to remove all orders.
