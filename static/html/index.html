API ENDPOINTS

Users Table

HTTP Methods Path Description
GET /user Retrieves a list of all users.
GET /user/:id Retrieves details of a specific user by ID.
POST /user/register Add a new user.
POST /user/login User is able to login.
PATCH /user/:id Updates details of a specific user by ID.
DELETE /user/:id Removes a user account by ID.

Foods Table

HTTP Methods Path Description
GET /hotel Retrieves a list of all hotels.
GET /hotel/recent Retrieves the 4 most recent products.
GET /hotel/:id Retrieves details of a specific hotel by ID.
POST /hotel/add Add a new hotel
PATCH /hotel/:id Updates details of a specific hotel by ID.
DELETE /hotel/:id Removes a hotel by ID.

Orders Table

HTTP Methods Path Description
GET /user/:uid/bookings Retrieves a list of all bookings.
GET /user/:uid/booking Retrieves all the bookings of a particular user.
POST /user/:uid/booking A user is able to add a booking.
PATCH /user/:uid/booking/:bookingID An admin is able to update booking.
DELETE /user/:uid/booking/:bookingID User is able to remove a single booking
DELETE /user/:uid/bookings User is able to remove all bookings

Stories Table

HTTP Methods Path Description
GET /story Retrieves a list of all reviews.
POST /story/addStory A user is able to add a review.
DELETE /story/:id Allows admin user to delete review.
